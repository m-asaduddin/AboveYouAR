#  AR Jump Game

This is Projects to showcase AR capabilities is iOS 15+. This is a mini game with 2 players. 
Each player must be collaborated and coordination to reach maximum point. Every player's must be jumping to collect coin. 
If any of the player's head hit the coin, point will be increased. If one of the player's head hit the bomb, the point decreased.


AR Type that used in this project is AR Face Tracking with World Tracking Enabled. 
The 3D model used in this project is self made (I made it) with Reality Composer Pro
